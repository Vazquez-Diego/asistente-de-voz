# 🎙️ LocalVoice Assistant

> **Tu asistente de voz inteligente que funciona 100% en tu máquina.** Sin APIs externas, sin suscripciones, sin preocupaciones de privacidad.

---

## ✨ Características Principales

| Característica | Descripción |
|---|---|
| 🏠 **100% Local** | Todo el procesamiento ocurre en tu dispositivo. Ningún dato de voz sale de tu computadora. |
| 🔒 **Privacidad Total** | Sin conexión a servidores de terceros. Tus comandos y conversaciones permanecen locales. |
| 🚀 **Bajo Consumo** | Optimizado para ejecutarse en CPU sin necesidad de GPU dedicada. |
| 🌐 **Multilingüe** | Soporte para español, inglés y otros idiomas mediante modelos locales. |
| ⚡ **Respuesta en Tiempo Real** | Latencia mínima gracias al procesamiento edge. |
| 🔧 **Extensible** | Sistema de plugins para añadir comandos personalizados. |
